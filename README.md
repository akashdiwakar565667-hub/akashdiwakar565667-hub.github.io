# 🚗 Ultimate Racing Game

A simple browser racing game made using **HTML, CSS and JavaScript**.

## 🎮 Features

* 3 Lane highway
* Player car control
* Enemy traffic cars
* Moving road animation
* Score system
* Speed increases with time
* Crash detection
* Start screen
* Mobile controls

## 🎮 Controls

**Keyboard**

* ⬅ Left Arrow → Move Left
* ➡ Right Arrow → Move Right

**Mobile**

* Left Button → Move Left
* Right Button → Move Right

## ▶ How to Run

1. Download the project
2. Open the file **ultimate_racing_game.html**
3. Run it in any browser (Chrome, Edge, etc.)
4. Click **Start Game**
